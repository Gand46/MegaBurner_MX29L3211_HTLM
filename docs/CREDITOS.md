# Créditos y agradecimientos

Este proyecto es una traducción/evolución de la parte de PC de MegaBurner hacia HTML/Web Serial. Se conservan los agradecimientos y referencias al trabajo original.

## Proyectos base

- **maximaas/MegaBurner**: proyecto original de programador MX29L3211 con Arduino Mega 2560, aplicación Java/SWT y documentación de hardware.
- **sanni/cartreader / Open Source Cartridge Reader**: base comunitaria de cart reader y lógica de programación flash de 16 bits utilizada por MegaBurner.
- **mkeller0815/MEEPROMMER**: referencia de comunicación PC ↔ Arduino Mega usada por MegaBurner.
- **Macronix**: datasheet y especificaciones de la MX29L3211.
- Comunidad retro-gaming y preservación de cartuchos.

## Imágenes usadas

Las imágenes `megaburner_hw.png` y `megaburner_window.png` fueron tomadas del repositorio original `maximaas/MegaBurner`, carpeta `pics`, y se incluyen como referencia y atribución visual al origen del proyecto.
