# Credits and Source Acknowledgements

This project is a Web Serial port and optimization branch built around the MegaBurner/MX29L3211 workflow.

## Upstream projects

- MegaBurner by maximaas: https://github.com/maximaas/MegaBurner
- Open Source Cartridge Reader / sanni cartreader: https://github.com/sanni/cartreader
- MEEPROMMER is credited by MegaBurner as the origin for PC-to-Mega2560 serial read/write communication.

## Reference documentation

- MX29L3211 datasheet: https://www.backoldgaming.com/data/medias/files/mx29f3211/MX29L3211.pdf

## Thanks

Thanks to the original authors and community projects that made this possible. This repository is intended as documentation of the porting path from the Java/SWT MegaBurner utility to a browser-based Web Serial application, while preserving the Arduino Mega and MX29L3211 programming logic.
