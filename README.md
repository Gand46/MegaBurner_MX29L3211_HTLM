# MegaBurner_HTLM
Port HTML/Web Serial de MegaBurner para programar y verificar memorias MX29L3211 con Arduino Mega 2560. Reemplaza la app Java/DLL por una interfaz web, conserva la lógica basada en MegaBurner/cartreader y añade FastWrite, FastRead, CRC rápido, UI XP, manual y documentación técnica.
