MegaBurner Web Serial v0.5B Baud1M Turbo

Port de la aplicación de PC de MegaBurner a HTML + JavaScript. v0.5B conserva FastRead, FastWrite, FastCRC y PageSize 256 nativo, pero sube el perfil recomendado de comunicación a 1.000.000 baudios. Usa esta versión solo con firmware compilado a 1.000.000. 500000 queda como fallback visual para volver a perfiles anteriores. Diseñado para MX29L3211 / 4 MB con Arduino Mega 2560.
