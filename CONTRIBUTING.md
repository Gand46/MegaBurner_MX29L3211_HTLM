# Contribuir

Las contribuciones son bienvenidas, especialmente en:

- Mejoras de documentación.
- Validaciones con diferentes placas Arduino Mega.
- Diseños de shield o adaptadores.
- Soporte documentado para memorias de 64 Mbit.
- Pruebas con chips compatibles.

Toda contribución debe mantener seguridad eléctrica: la MX29L3211 es de 3.3 V.
